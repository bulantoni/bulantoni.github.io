# Структура

## Блок инструкций

В JS блоком инструкций называют все находится внутри фигурных скобок `{ }`.

```js
{
  ;
  7 + 9;  //Блок инструкций
  ;;;
}
```
В зависимости от контекста, после блока инструкций, не ставиться точка с запятой.

## Комментарии

```js
3 + 2; //Комментарий

/*
Комментарий
*/
```
